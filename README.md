# 🧬 نظام SOL Data Agent (وكيل بيانات SOL)

![SOL Data Agent Concept Image](https://cdn-icons-png.flaticon.com/512/3094/3094896.png)

## 📌 الفكرة العامة
نظام **SOL Data Agent** هو نظام ذكي لتنظيف البيانات يعمل بشكل ذاتي تماماً ومُدعوم بالذكاء الاصطناعي (AI). مهمته الأساسية هي سد الفجوة بين مجموعات البيانات الخام الفوضوية وغير المكتملة أو غير المترابطة، وتحويلها إلى بيانات نقية جاهزة للإنتاج.

على عكس السكربتات اليدوية المعتادة، يعمل وكيل SOL بشكل متسلسل عبر 4 مراحل متميزة:
1. **التحليل والمسح (Analysis & Scanning)**: استخراج البيانات الوصفية (metadata)، وتصنيف الأعمدة، وتحديد الشذوذيات بانسيابية تامة عبر مجموعة هائلة من صيغ الملفات وأنواع قواعد بيانات SQL.
2. **توليد خطة التنظيف (Strategy Generation)**: استخدام نموذج لغوي ضخم (LLM مثل Llama-3) لقراءة وفهم سياق البيانات المفقودة وصياغة الاستراتيجية الأمثل والأكثر ذكاءً للتعامل معها.
3. **التنفيذ (Execution)**: تنظيف البيانات بكفاءة عن طريق التركيز على القيم الاكرارية، القيم المفقودة (Nulls)، الأنواع الخاطئة للبيانات، تنسيق الأرقام والهواتف والبريد الإلكتروني، وكذلك التعامل مع القيم المتطرفة (Outliers).
4. **التحقق والتصدير (Validation)**: تقديم مقارنة مرئية جنباً إلى جنب للبيانات قبل وبعد التنظيف عبر واجهة Streamlit تفاعلية وتصميم مستلهم من الخيال العلمي (Sci-Fi)، مع إمكانية تصدير البيانات المطوّرة على الفور أو كتابتها وحفظها بشكل مباشر في قاعدة البيانات الأصلية.

## 🚀 المميزات الرئيسية (تحديثات النواة والمنطق العقلي)
- **محمل البيانات الشامل (Universal Data Loader)**: يعالج حزمة هائلة من أنواع الملفات المختلفة بشكل تلقائي - بما في ذلك `CSV`, `Excel`, `JSON`, `XML`, `Parquet`, `Feather`, `HDF5 / H5`, و `ORC`.
- **تصدير ذكي بالصيغة الأصلية (Dynamic Extension Persistence)**: يقوم محرك التصدير الآن تلقائياً بإرجاع البيانات المنظفة بنفس الامتداد والصيغة المعقدة للملف الأصلي المرفوع.
- **موصّلات مباشرة لقواعد البيانات (Direct SQL Connectors)**: قادر على الوصول، استيراد الجداول، القراءة والتحديث المباشر للبيانات (`Sync`) داخل قواعد البيانات من نوع `SQLite`, `MySQL`, و `PostgreSQL` بطريقة آمنة.
- **توجيه التنظيف بالأهداف (User-Defined Goals)**: يتيح النظام الآن للمستخدم كتابة هدف بلغة طبيعية (مثل: "قم بتحويل أرقام الهواتف للصيغة الدولية" أو "تجاهل تنظيف النصوص وركز على الخسائر الرقمية")، ليقوم النواة بحقن هذا الهدف لتعديل استراتيجية التنظيف جذرياً.
- **معاينة الاستراتيجية المسبقة (Dynamic Strategy Previews)**: يقوم النظام ببناء وتوليد ملخصات تقنية مقروءة بوضوح (وعرض هيكل JSON) لكل خطة تنظيف (Alpha, Beta, Gamma) وتأثيرها على الأعمدة بدقة، *قبل* تنفيذ عملية التنظيف.
- **ذكاء اصطناعي استراتيجي (AI-Powered Strategist)**: يحلل الذكاء الاصطناعي السياق الكامل للبيانات التالفة ويقرر *بدقة* خطة تعويض القيم المفقودة أو تصحيح النصوص بناءً على لغويات المنطق الدقيق.

## 🧩 الأنظمة الفرعية والأدوات المدمجة (Subsystems & Tools)
إلى جانب نواة التنظيف الأساسية، يحتوي النظام على ترسانة من الأدوات التي تجعله منصة بيانات شاملة لحل المشاكل من الجذور:
- **محرك التخطيط الدلالي الثنائي (Semantic Binary Mapping):** بمجرد إرسال الملف، تكتشف هذه الأداة الذكية الأعمدة والقيم المنطقية من سياقها (مثل "موافق/مرفوض" أو "Yes/No") وتُجري عملية Mapping مباشرة لتحويلها إلى (0 و 1) بدقة عالية، مما يجعل البيانات جاهزة فوراً لخورازميات الذكاء الاصطناعي.
- **محرك استخراج النصوص (OCR Engine):** أداة مدمجة مدعومة بالذكاء الاصطناعي لاستخراج النصوص والبيانات من الصور والمستندات بذكاء وتحويلها لبيانات قابلة للمعالجة والتنظيف كـ DataFrame.
- **نماذج البيانات (SOL Forms):** نظام متصل متكامل (كبديل متقدم لـ Google Forms) لإنشاء نماذج وتطبيقات بصرية لجمع البيانات المباشرة من المستخدمين وتخزينها، مما يمنع المشاكل من مصدرها.
- **معمل الذكاء الاصطناعي ومحاكي التلف (AI Lab / Corruptor):** بيئة اختبار (Sandbox) قوية لإجراء تجارب ومحاكاة أعطال وتخريب مدروس للبيانات لاختبار جودة وموثوقية نماذج التنظيف.
- **تقارير التدقيق (Audit Report):** لوحة لتوليد تقارير وتدقيقات شاملة لمراجعة وتأكيد كل تدخل قام به الذكاء الاصطناعي، والتغييرات التي طرأت على البيانات لضمان الشفافية المؤسسية.

## 💎 أهمية نظام SOL Data Agent
جميع مهندسي وعلماء ومحللي البيانات يجمعون دائماً على مشكلة شائعة جداً: **تنظيف البيانات يستهلك تقريباً 80% من وقت أي مشروع برمجي أو بيانات.**

نظام **SOL Data Agent** يغيّر هذا المفهوم جذرياً.
- **مُسرّع هائل للإنتاجية**: أتمتة تصحيح الأخطاء والقيم المجهولة، مطابقة النصوص غير المنتظمة، توحيد تنسيق التواريخ، وحل المشاكل الرقمية بشكل كامل وذكي، يقلص ما كان يستغرق "أياماً" إلى مجرد "لحظات وثوانِ معدودة".
- **تقليل الأخطاء البشرية والانحياز**: التعديل اليدوي عرضة لسقوط البيانات أو تلفها. بينما وجود خوارزميات ذكاء اصطناعي تعتمد على الثقة المزدوجة (Dual-Confidence) يضمن عدم تنفيذ أي خطة دون تقييمها منطقياً لتلافي التحويلات السيئة.
- **التفاعل السلس مع قواعد البيانات**: تقديم روابط سريعة ومباشرة لقواعد بيانات المؤسسة، يقدم أفضل حماية وأسرع تعامل لعمليات التنظيف المعقدة بشكل لحظي وبلا الحاجة لتنزيل وإعادة تخزين ملفات هشة برمجياً.

## 💰 كيف تربح من هذا المشروع؟
يشتمل هذا النظام على قيمة تجارية وعملية جبّارة! وفيما يلي استراتيجيات مقترحة لتحقيق ربح قوي ومستمر منه:

### 1. **تقديمه كبرمجية كخدمة - (SaaS)**
قم باستضافة هذه الواجهة الرائعة (Streamlit) عبر خوادم سحابية مثل AWS، Render، أو Streamlit Cloud. وقسّم الخدمة إلى خطط تناسب الشركات المختلفة وأصحاب الأعمال:
- **الباقة المجانية (Free Tier):** ملفات CSV صغيرة الحجم (تصل لـ 5 ميجابايت).
- **باقة المحترفين ($25/شهرياً):** رفع ملفات غير محدودة بكل الصيغ (Feather/HDF5..الخ) مع قدرة الربط بقاعدة بيانات SQL واحدة.
- **باقة الشركات ($500/شهرياً):** تزامن لا محدود لقواعد بيانات SQL، استضافة معزولة ومنفصلة لضمان الخصوصية وسرية البيانات، وتوجيه الذكاء الاصطناعي لمهام محددة تتعلق بمقاييس الشركة (KPIs).

### 2. **تراخيص المؤسسات و(On-Premise Installations)**
الشركات الكبرى والأمنية لا تفضل تسريب أي شكل من أشكال البيانات لخوادم خارجية سحابية. يمكنك بيع ترخيص مؤسسي لتثبيت واجهة SOL Data Agent بالكامل داخلياً (على سيرفراتهم الخاصة والمحمية بجدار الحماية لديهم) وبما يطابق لوائح وقوانين الخصوصية، مع زيادة الرسوم استناداً لعدد مقاعد الاستخدام السنوية المحددة بالترخيص.

### 3. **محرك عمل حر صاروخي (Supercharged Freelancing)**
إذا كنت تعمل بشكل مستقل كمهندس بيانات أو مبرمج في منصات مثل Upwork أو Fiverr، يمكنك حصد مشاريع "تنظيف وتنظيم البيانات - Data Cleaning". باستخدامك لنظام وكيل SOL، سيستغرق إنجاز أي عمل، كان سيستغرق في الظروف العادية 5 أيام من التدقيق الممل، بضعة دقائق فقط بنتائج دقيقة ومبهرة وتفوق توقعات العميل! وبذلك ستتمكن من مضاعفة الإنتاجية وجني أرباح رهيبة بدون جهد!

### 4. **واجهة برمجة التطبيقات (API-First)**
افصل منطق النظام الخلفي للذكاء الاصطناعي والـ (API/FastAPI) كمنصة خدمات برمجية. الأنظمة والمواقع والبرمجيات الأخرى، التي تحتاج إلى وظائف تنظيف بيانات قوية بلمح البصر، يمكنها إرسال طلبات (REST API) لخادمك الخاص، وتقوم أنت بفرض رسوم أو باقات مقابل كل نقطة استخدام ذكية يتم استهلاكها (Charging per API Request/Token).

## 🛠️ التثبيت وطريقة التشغيل (Installation & Usage)

### 1. المتطلبات الأساسية
تأكد من وجود بيئة `Python 3.10+` وقم بتحميل مستودع المشروع.
سوف تحتاج أيضاً إلى مفتاح API لمكتبات الذكاء (مثل GROQ) ويتم ضبطها داخل ملف `app.py`.

### 2. تثبيت الحزم والمتطلبات
```bash
pip install -r requirements.txt
```
*(ويمكنك إضافة المكتبات الخاصة للتعامل المتقدم لقواعد البيانات والملفات الشاملة في حال رغبت بتجربتها):*
```bash
pip install pymysql psycopg2-binary pytables sqlalchemy pyarrow
```

### 3. تشغيل النظام
شغل الواجهة التفاعلية محلياً والمطورة باستخدام أداة Streamlit عن طريق الأمر التالي:
```bash
python -m streamlit run app.py
```

### 4. تشغيل الاختبارات
لضمان وتوثيق كفاءة جميع وظائف التصدير والاستيراد، وتحمل ملفات الأخطاء أو الصيغ غير المدعومة:
```bash
python -m pytest tests
```
