STAGE 01

Analysis

100%

Columns Profiled

SOL's analyzer scans your dataset end-to-end before touching a single value. It extracts schema, infers semantic types, maps missing values, and flags statistical anomalies — building a complete 'data profile' used by all subsequent stages.

  • Auto-detect 7+ file formats
  • Column semantic classification (email, phone, date…)
  • Missing value density & pattern mapping
  • Duplicate row detection
  • Outlier flagging via IQR & Z-score
psychology

STAGE 02

Strategize

< 2s

Planning Time

The AI Strategist (powered by Llama-3 via Groq API) reads the data profile and constructs a tailored cleaning plan — choosing between Alpha (gentle), Beta (balanced), or Gamma (aggressive) strategies based on data characteristics and your natural-language goal.

  • 3 plan tiers: Alpha / Beta / Gamma
  • User goal injection via plain English
  • JSON strategy preview before execution
  • Dual-confidence scoring for every decision
  • Column-level action mapping
auto_fix_high

STAGE 03

Execution

10

Core Modules

SmartDataCleaner executes the approved strategy precisely — imputing missing values using statistical methods, removing outliers, standardizing date formats, fuzzy-matching text inconsistencies, and eliminating duplicates — all in a single efficient pass.

  • Null imputation (mean, median, mode, smart)
  • Z-score & IQR outlier removal
  • ISO 8601 date standardization
  • Fuzzy text normalization for categories
  • Exact & near-duplicate elimination
verified

STAGE 04

Validation

0.3s

Time to Export

A complete before-vs-after comparison with cell-level diff highlighting shows exactly what changed and why. The health score quantifies improvement. Export in any format or write-back directly to your SQL database — no intermediate files.

  • Side-by-side comparison grid
  • Cell-level diff map highlighting
  • Health score % improvement
  • Smart export in original format
  • Direct PostgreSQL / MySQL / SQLite sync
