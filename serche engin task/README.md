# 🤖 AI Dataset Advisor - Intelligent Search 🤖
### مستشار البيانات الذكي ومحرك البحث الهجين لمجموعات بيانات الذكاء الاصطناعي (عربي / إنجليزي)

---

## 📌 فكرة المشروع (Project Concept)

يمثل مشروع **AI Dataset Advisor** منصة ذكية متكاملة ومحرك بحث هجين صُمم خصيصاً لمساعدة مهندسي البيانات ومطوري الذكاء الاصطناعي في العثور على مجموعات البيانات (Datasets) المناسبة لمشاريعهم وتطبيقاتهم البرمجية بشكل فوري. 

بدلاً من قضاء ساعات طويلة في البحث اليدوي في المنصات العالمية مثل Kaggle، يتيح هذا النظام للمستخدم كتابة وصف باللغة الطبيعية (باللغة العربية أو الإنجليزية) يوضح طبيعة مشروعه أو الفكرة التي يريد تطبيقها (مثل: *"أريد داتا صغيرة لتصنيف نصوص المشاعر باللغة العربية"* أو *"I need a dataset for house price prediction in the USA under 10000 rows"*). 

يقوم النظام تلقائياً بتحليل نية الاستعلام واستخراج المعايير المطلوبة، ثم يقوم برحلة بحث وتصفية هجينة ومتقاطعة تجمع بين البيانات المخزنة محلياً والبحث الفوري المباشر عبر Kaggle API، مروراً بنظام تقييم دلالي متقدم لترتيب النتائج، لينتهي بتقديم أفضل مجموعات البيانات متبوعة بـ **تبرير ذكي ومخصص مولد بواسطة الذكاء الاصطناعي (LLM-based reasoning)** يوضح للمستخدم بدقة سبب اختيار هذه البيانات ومناسبتها لهدفه.

---

## 🚀 أبرز المميزات والخصائص الفنية

1. **فهم نية الاستعلام الذكي (Intent Parsing & Query Understanding):**
   * عند إدخال الاستعلام، يقوم النظام بتحليله عبر LLM لاستخراج القيود والمواصفات بدقة (نوع المهمة: تصنيف، انحدار، معالجة لغة طبيعية، رؤية حاسوبية، واللغة المستهدفة، والحد الأدنى والأقصى لعدد الصفوف المطلوبة).
   
2. **البحث الهجين ثنائي المسار (Hybrid Retrieval Flow):**
   * **مسار Kaggle المباشر:** جلب فوري لأحدث مجموعات البيانات ذات الجودة العالية مباشرة عبر Kaggle API باستخدام الاستعلام الأساسي مع إمكانية التوسع التلقائي للطلبات (Expansion Queries) لضمان تغطية كاملة.
   * **مسار قاعدة البيانات المحلية (Local Fallback Seeding):** في حال انقطاع الإنترنت أو عدم وجود نتائج مباشرة، يرتد النظام تلقائياً إلى قاعدة بيانات بذور محلية عالية الجودة كخيار احتياطي ذكي.

3. **دعم كامل للغة العربية (Arabic Native Support & RTL):**
   * كشف تلقائي للغة الاستعلام (عربي/إنجليزي).
   * واجهة مستخدم استثنائية تدعم اتجاه الكتابة من اليمين إلى اليسار (RTL) للنصوص والتبريرات العربية لضمان تجربة مستخدم سلسة ومريحة للمتحدثين بالعربية.
   * توليد تبريرات باللغة العربية الفصحى وبأسلوب احترافي عند الاستعلام بالعربية.

4. **نظام الترتيب الذكي الموزون (Advanced Hybrid Reranking & Scoring):**
   يتم حساب درجة مطابقة هجينة وعلمية لكل مجموعة بيانات وتصنيفها من **0% إلى 100%** بناءً على المعادلة الموزونة التالية:
   * **40% تطابق دلالي (Semantic Alignment):** حساب التشابه الجيبي (Cosine Similarity) بين متجه الاستعلام ومتجه تفاصيل مجموعة البيانات.
   * **20% توافق نوع المهمة (ML Task Fit):** مدى مطابقة المهمة (مثلاً Regression مع توقع الأسعار).
   * **15% توافق الوسيط (Modality Fit):** مطابقة نوع البيانات (نصية، جدولية، صور).
   * **15% توافق اللغة (Language Alignment):** توافق لغة البيانات مع اللغة المطلوبة.
   * **10% توافق الحجم (Row Constraints Compliance):** تقييم عقابي تدريجي مرن في حال تجاوز عدد الصفوف المطلوبة أو نقصانها.

5. **الكاش الدلالي (Semantic Cache):**
   * لتجنب تأخير البحث وتكرار استدعاء الـ APIs لطلبات البحث المتشابهة دلالياً، يقوم النظام بالاحتفاظ بنتائج البحث السابقة ذكياً وتوفيرها فوراً عند تكرار الفكرة نفسها.

6. **لوحة تحكم وتدقيق تفاعلية (Diagnostics Sidebar):**
   * متابعة لحظية لحالة اتصال قاعدة البيانات المتجهة (Qdrant).
   * عرض إحصائيات لعدد النقاط المتجهة المسجلة (Vector Points Count).
   * إمكانية تغيير النموذج اللغوي النشط (Active LLM) بضغطة زر.
   * تقديم اقتراحات بحث جاهزة ثنائية اللغة لتسهيل بدء التجربة.

## 🛠️ التقنيات المستخدمة (Technology Stack)

* **الواجهة الأمامية (Frontend):** 
  * [Streamlit](https://streamlit.io/) لبناء واجهة تفاعلية فورية ومصممة بأسلوب عصري مخصص (Glassmorphic Container, Metallic Gradients, RTL Styling, metadata pills).
* **الخلفية (Backend):**
  * [FastAPI](https://fastapi.tiangolo.com/) لبناء API ذكي ومرن ومتكامل مدعوم بـ Lifespan Events للتهيئة الذاتية والتغذية التلقائية لقواعد البيانات.
* **قواعد البيانات (Databases):**
  * **Relational Database:** [SQLite](https://www.sqlite.org/) مدعومة بمحرك الاستعلام غير المتزامن [aiosqlite] عبر [SQLAlchemy ORM] لإدارة وتخزين سجلات البحث وتوصيات المستشار وحفظها للتحليل اللاحق.
  * **Vector Database:** [Qdrant](https://qdrant.tech/) لإدارة البحث الدلالي عالي الأداء مع التخزين المحلي التلقائي دون الحاجة لإعدادات معقدة.
* **نماذج الذكاء الاصطناعي (AI & Embeddings):**
  * **توليد المتجهات:** نموذج `sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2` لتوليد تضمينات (Embeddings) عابرة للغات تدعم العربية والإنجليزية بكفاءة متناهية.
  * **توليد النصوص والاستنتاج:** نماذج [Groq Cloud] المتفوقة والمنخفضة التأخير (`llama-3.3-70b-versatile` / `llama-3.1-8b-instant`).

## ⚙️ التثبيت والتشغيل (Setup & Running)

### 1. إعداد متغيرات البيئة
قم بإنشاء ملف `.env` في المجلد الرئيسي للمشروع (وهو مجهز بالفعل) وتأكد من توفر المفاتيح التالية:
```env
KAGGLE_API_KEY=YOUR_KAGGLE_API_KEY
KAGGLE_USERNAME=YOUR_KAGGLE_USERNAME
GROQ_API_KEY=YOUR_GROQ_API_KEY
GROQ_MODEL=llama-3.1-8b-instant
```

> [!NOTE]
> للحصول على مفتاح Kaggle، يمكنك الدخول إلى حسابك على منصة Kaggle وتوليد رمز API جديد (Kaggle API Token) من إعدادات حسابك.

---

### 2. التشغيل باستخدام Docker Compose (الطريقة الموصى بها 🐳)
لتشغيل المشروع بالكامل بحاوية واحدة دون الحاجة لتثبيت بايثون أو المكتبات محلياً:
```bash
docker-compose up --build
```
سيكون خادم FastAPI جاهزاً على المنفذ `8000`.

---

### 3. التشغيل اليدوي المحلي (Manual Setup 🐍)

#### أ. تشغيل FastAPI Backend
1. قم بإنشاء بيئة وهمية وتفعيلها:
   ```bash
   python -m venv venv
   # لتفعيل البيئة على نظام ويندوز:
   venv\Scripts\activate
   ```
2. قم بتثبيت المكتبات المطلوبة:
   ```bash
   pip install -r backend/requirements.txt
   ```
3. ابدأ تشغيل خادم FastAPI:
   ```bash
   uvicorn backend.app.main:app --reload --port 8000
   ```

#### ب. تشغيل Streamlit Frontend
في نافذة طرفية (Terminal) أخرى نشطة، قم بتشغيل واجهة المستخدم لـ Streamlit:
```bash
streamlit run streamlit_app.py
```
ستفتح واجهة الويب تلقائياً على المتصفح عبر الرابط `http://localhost:8501`.

## 📊 قاعدة البيانات والنماذج الهيكلية (Database Schemas)

1. **Dataset:** يمثل جدول مجموعات البيانات المخزنة محلياً أو المجلوبة من كاجل مع تفاصيل الحجم والتقييم والتصنيف ورابط المصدر.

3. **Recommendation:** يسجل توصيات المستشار الذكي ودرجة مطابقتها مع التبرير المخصص لكل عملية بحث لمزيد من تدقيق وتحليل الجودة.
